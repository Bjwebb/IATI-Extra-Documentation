

Example Usage
~~~~~~~~~~~~~

Example:

.. code-block:: xml

    <location ref="">
      ...
      
      <point srsName="http://www.opengis.net/def/crs/EPSG/0/4326">
        <coordinates>31.616944 65.716944</coordinates>
      </point>
      
      ...
    </location>

NOTE: The @srsName currently should always be http://www.opengis.net/def/crs/EPSG/0/4326
