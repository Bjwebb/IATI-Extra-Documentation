Example Usage
~~~~~~~~~~~~~
Example declaring just the ``DocumentCategory`` code *A01*:

.. code-block:: xml

      <category code="A01"/>

Full example:

.. code-block:: xml

    <document-link format="application/vnd.oasis.opendocument.text" url="http:www.example.org/docs/report_en.odt">
      <category code="A01"/>
	  <language code="en"/>
      <title>Project report 2013</title>
    </document-link>
