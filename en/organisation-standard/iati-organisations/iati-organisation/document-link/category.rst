Example Usage
~~~~~~~~~~~~~
The code should be declared in any usage of this element:

.. code-block:: xml

        <category code="B01"/>

Full example:

.. code-block:: xml

    <document-link format="application/vnd.oasis.opendocument.text" url="http:www.example.org/docs/report_en.odt">
      <category code="B01"/>
	  <language code="en"/>
      <title>Annual report 2013</title>
    </document-link>
